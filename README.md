# HighSpeedRail_GTFS

港鐵[高速鐵路](https://www.highspeed.mtr.com.hk/) [GTFS](https://gtfs.org/)，仅包括“動感號”担当车次

### 功能

- [x] [楼层（待改进）](https://gtfs.org/getting-started/features/pathways/#levels)
- [x] [线路颜色](https://gtfs.org/getting-started/features/base-add-ons/#route-colors)
- [x] [水牌](https://gtfs.org/getting-started/features/base-add-ons/#headsigns)
- [x] [站点无障碍](https://gtfs.org/getting-started/features/accessibility/#stops-wheelchair-accessibility)
- [x] [行程无障碍](https://gtfs.org/getting-started/features/accessibility/#trips-wheelchair-accessibility)
- [x] [月台编号（福田、光明城、虎门、南沙北）](https://gtfs.org/getting-started/features/base/#stops)
- [x] [出入口（仅福田，待完善）](https://gtfs.org/getting-started/features/base-add-ons/#location-types)
- [x] [车次（仅港鐵動感號担当）](https://gtfs.org/getting-started/features/base/#trips)
- [x] [时刻表（仅港鐵動感號担当，精确到分钟）](https://gtfs.org/getting-started/features/base/#stop-times)
- [x] [服务日期（缺少例外，某些情况下有效期可能短过七日](https://gtfs.org/getting-started/features/base/#service-dates)
- [x] [路线](https://gtfs.org/getting-started/features/base/#routes)
- [x] [路径](https://gtfs.org/getting-started/features/base-add-ons/#shapes)
